# Churn Sense AI

## Project Description

Churn Sense AI is a web application that helps businesses predict customer churn using data analysis and machine learning techniques. The system analyzes customer data and provides insights to identify customers who are likely to stop using a service. This helps organizations take proactive steps to retain customers and improve customer satisfaction.

## Features

* Predicts customer churn based on input data
* Allows users to upload customer datasets
* Provides insights about customer behavior
* Displays results through an interactive dashboard
* Easy-to-use user interface for analysis

## Technologies Used

* React
* Vite
* TypeScript
* Tailwind CSS
* Machine Learning concepts

## Project Structure

src/ – Contains all React components and application logic
public/ – Static files used in the project
package.json – Project dependencies and scripts
vite.config.ts – Configuration file for Vite

## How to Run the Project

1. Clone the repository
   git clone https://github.com/sudeeksha-05/Churn-sense-ai.git

2. Navigate to the project folder
   cd Churn-sense-ai

3. Install the required dependencies
   npm install

4. Start the development server
   npm run dev
